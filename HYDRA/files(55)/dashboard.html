Core Features

Every component of the Hydra surveillance stack — running in real-time on your hardware.

YOLOv8 Detection

Real-time person and weapon detection using YOLOv8s with custom-trained weapon weights. Falls back to HOG detector if GPU unavailable.

Behaviour Analysis

Detects robbery (weapon inside person box 2s+), fighting (IOU > 0.45 for 3s+), and loitering (stationary 15s+) with zero false alarms.

Fire Detection

HSV-based fire detector with 4-frame history confirmation. Detects fire regions, draws bounding boxes, and triggers alert with beep + SMS.

Face Blur / Privacy

Haar cascade face detection blurs all faces in USER mode for GDPR compliance. ADMIN mode shows faces with bounding boxes. Toggle with key A.

Cross-Camera Tracking

HSV histogram appearance matching assigns global IDs across cameras. Shows "Also seen: CAM2" labels. Database persisted to cross_camera_db.json.

SMS Alerts (Gmail)

No Twilio needed. Sends SMS via Gmail SMTP + carrier email-to-SMS gateway (Jio, Airtel, Vi). 60s cooldown prevents spam. Runs in background thread.

How It Works

A full pipeline from camera input to threat alert — all running locally on your machine.

01 — Capture

Supports webcam, video file, and mobile phone camera (IP Webcam app). MobileCameraSource reads frames in a background thread with buffer flushing to prevent lag. Auto-reconnects on stream loss.

02 — Detect

YOLOv8s runs in a dedicated YOLOThread — detection never blocks the display loop. Frames are submitted at 416×416, results scaled back to display resolution. HOG fallback if YOLOv8 is unavailable.

03 — Analyse

BehaviourAnalyser checks every frame for robbery (weapon inside person), fighting (IOU overlap timer), and loitering (centroid stationary timer). FireDetector uses 4-frame confirmation to eliminate false positives.

04 — Score

ThreatAnalyzer scores 0–100%: weapon alone = 60%, weapon + person = 90%, multiple weapons = 100%, fire = 85%, behaviour events from 40–95%. Alert triggers at score ≥ 50%.

05 — Alert

On threat: beep sound plays (3× via winsound/afplay/terminal bell), SMS sent via Gmail SMTP to configured phone number with timestamp, camera ID, threat type, and score. 60s cooldown between SMS alerts.

06 — Display

HUDRenderer draws all bounding boxes, behaviour labels, cross-camera IDs, threat bar, FPS, face/weapon/fire counts, and flashing alert banner — all on a single OpenCV window at full HD.

class SMSNotifier:
    GMAIL_USER         = "your.gmail@gmail.com"
    GMAIL_APP_PASSWORD = "xxxx xxxx xxxx xxxx"
    RECIPIENT_EMAIL    = "10digitnumber@jio.com"

    def send(self, threat_type, threat_score, camera_id):
        threading.Thread(target=self._send_sms,
            args=(threat_type, threat_score, camera_id), daemon=True).start()

    def _send_sms(self, threat_type, threat_score, camera_id):
        import smtplib
        from email.mime.text import MIMEText
        body = f"HYDRA ALERT\nCamera:{camera_id}\n"
               f"Threat:{threat_type}\nScore:{threat_score}%"
        msg = MIMEText(body)
        msg["Subject"] = f"ALERT: {threat_type}"
        msg["From"] = self.GMAIL_USER
        msg["To"]   = self.RECIPIENT_EMAIL
        with smtplib.SMTP_SSL("smtp.gmail.com", 465) as s:
            s.login(self.GMAIL_USER, self.GMAIL_APP_PASSWORD)
            s.sendmail(self.GMAIL_USER, self.RECIPIENT_EMAIL, msg.as_string())

class FireDetector:
    def __init__(self): self._history = deque(maxlen=4)
    def detect(self, frame):
        hsv  = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
        m1   = cv2.inRange(hsv, np.array([0,200,200]),  np.array([20,255,255]))
        m2   = cv2.inRange(hsv, np.array([165,200,200]),np.array([180,255,255]))
        mask = cv2.bitwise_or(m1, m2)
        k    = cv2.getStructuringElement(cv2.MORPH_ELLIPSE,(7,7))
        mask = cv2.morphologyEx(mask, cv2.MORPH_OPEN, k, iterations=2)
        h,w  = frame.shape[:2]
        ratio = cv2.countNonZero(mask)/(h*w)
        self._history.append(ratio > 0.015)
        if len(self._history)==4 and all(self._history):
            cnts,_ = cv2.findContours(mask,cv2.RETR_EXTERNAL,cv2.CHAIN_APPROX_SIMPLE)
            boxes  = [(x,y,x+bw,y+bh) for c in cnts
                      if cv2.contourArea(c)>3000
                      for x,y,bw,bh in [cv2.boundingRect(c)]]
            return boxes, ratio
        return [], 0.0
